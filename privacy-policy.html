Privacy Policy for AI Smart Translator Pro

Last Updated: October 27, 2025

Introduction

AI Smart Translator Pro ("we", "our", or "the extension") is committed to protecting your privacy. This Privacy Policy explains how we handle information when you use our Chrome extension.

Information We Collect

Information Stored Locally

AI Smart Translator Pro stores the following information locally in your browser using Chrome's storage API:

  1. API Keys
    • Google Gemini API key (if you choose to use Gemini)
    • These are stored encrypted in your browser's local storage
    • Never transmitted to our servers (we don't have servers)
  2. User Preferences
    • AI provider selection (Gemini or Ollama)
    • Model preferences
    • Custom translation prompts
    • Server URLs (for local Ollama)
  3. No Personal Data
    • We do NOT collect, store, or transmit any personal information
    • We do NOT track your browsing history
    • We do NOT collect analytics or usage statistics

How We Use Information

Text Processing

When you use the translation or grammar checking features:

  1. With Google Gemini:
    • Selected text is sent directly to Google's Gemini API
    • Processed according to Google's Privacy Policy
    • We do not intercept or store this data
  2. With Local Ollama:
    • All text processing happens locally on your machine
    • No data leaves your computer
    • Complete privacy and offline capability

API Keys

  • API keys are stored locally in your browser only
  • Never transmitted to any third-party servers
  • You can delete them at any time through the extension settings

Data Sharing

We do NOT share any data with third parties because:

  • We don't collect any data ourselves
  • We don't have servers or databases
  • The extension operates entirely client-side

Third-Party Services

If you choose to use Google Gemini:

  • Your text is sent directly to Google's API
  • Subject to Google's Privacy Policy
  • Google's API Terms of Service

Data Security

  • All settings and API keys are stored using Chrome's secure storage API
  • No data is transmitted over unencrypted connections
  • Local Ollama option provides complete privacy

Your Rights

You have full control over your data:

  1. Access: View all stored settings in the extension options
  2. Modify: Change or update any settings at any time
  3. Delete: Remove the extension to delete all stored data
  4. Export: Export your settings as JSON backup

Children's Privacy

AI Smart Translator Pro does not knowingly collect any information from children under 13. The extension is designed for general audiences and does not require any personal information.

Changes to This Policy

We may update this Privacy Policy from time to time. Changes will be reflected in the "Last Updated" date at the top of this policy.

Permissions Explanation

The extension requests the following permissions:

  1. activeTab: To access selected text on the current webpage for translation
  2. storage: To save your settings and preferences locally
  3. Host permissions (localhost): To connect to local Ollama server (optional)

Compliance

This extension complies with:

  • Chrome Web Store Developer Program Policies
  • GDPR (General Data Protection Regulation)
  • CCPA (California Consumer Privacy Act)

Summary

AI Smart Translator Pro respects your privacy. We don't collect, store, or share your personal data. Your settings and API keys stay on your device. When using cloud AI (Gemini), data is sent directly to that provider under their privacy policy.
